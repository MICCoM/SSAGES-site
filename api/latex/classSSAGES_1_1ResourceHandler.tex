\doxysubsection{Detailed Description}
Class that handles \mbox{\hyperlink{namespaceSSAGES}{S\+S\+A\+G\+ES}} resources for a simulation. 

\mbox{\hyperlink{classSSAGES_1_1ResourceHandler}{Resource\+Handler}} is responsible for intialzing the major components of a simulation including the engine, methods, and collective variables. It is also responsible for the lifetime of the objects it creates.

Each simulation engine must implement a driver which calls this resource handler and passes it the appropriate hook.
