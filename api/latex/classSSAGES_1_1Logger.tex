\doxysubsection{Detailed Description}
Base class for logging \mbox{\hyperlink{namespaceSSAGES}{S\+S\+A\+G\+ES}} data. 

The base class for logging useful data in \mbox{\hyperlink{namespaceSSAGES}{S\+S\+A\+G\+ES}} that is not necessarily written out by methods. Primarily this includes the value(s) of the collective variable(s) on the various walkers over time, the magnitude of the bias if a method supports it, and so on.
