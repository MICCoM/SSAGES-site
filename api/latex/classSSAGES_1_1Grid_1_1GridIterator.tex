\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
template$<$typename R$>$\newline
class S\+S\+A\+G\+E\+S\+::\+Grid$<$ T $>$\+::\+Grid\+Iterator$<$ R $>$}

Custom Iterator. 

This iterator is designed of travesing through a grid. The starting point is at grid index 0 for each dimension. The last valid grid point has the num\+\_\+points in each dimension, where num\+\_\+points is the number of grid points in the respective dimension.

The iterator can be used as a standard iterator with operator$\ast$ accessing the grid point at which the iterator currently is.

Additionally, the functions \mbox{\hyperlink{classSSAGES_1_1Grid_1_1GridIterator_a9d36203d74c669af7b389a6447d03d7b}{Grid\+Iterator\+::indices()}} and \mbox{\hyperlink{classSSAGES_1_1Grid_1_1GridIterator_ac7050053f8da7e4075d847c5f1abd9be}{Grid\+Iterator\+::coordinates()}} are provided. These functions return the indices of the current grid point and the center of the grid point interval in real space, respectively.

The iterator can be moved to an arbitrary position. As \mbox{\hyperlink{classSSAGES_1_1Grid_1_1GridIterator_a9d36203d74c669af7b389a6447d03d7b}{indices()}} returns a reference (and not a const reference), it can be used to move the iterator. For example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{HistIterator it = hist-\/>begin();}
\DoxyCodeLine{it.indices() = \{1,1,1\};}
\end{DoxyCode}


moves the iterator to the grid point \mbox{[}1, 1, 1\mbox{]}.

The iterator can be traversed in a standard fashion with the increment and decrement operators operator++ and operator-\/-\/. When the increment operator is invoked, the bin index for the lowest dimension is increased by 1. If it moves beyond the valid range in this dimension, the index is reset to 0 and the index of the next higher dimension is increased by 1. The decrement operator traveses the grid in the same fashion but opposite direction.

Additionaly, the iterator can be shifted by adding or subtracting a vector of ints. The vector needs to have the same dimension as the histogram.
